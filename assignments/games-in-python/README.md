
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic Hangman word-guessing game using Python. This assignment practices string manipulation, loops, conditionals, and basic I/O.

## 📝 Tasks

### 🛠️ Implement the core game loop

#### Description
Create a playable Hangman game that selects a word and lets a player guess letters until they either reveal the word or run out of attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept single-letter guesses and update the displayed progress (e.g., _ a _ _ m a n)
- Validate input (letters only, not previously guessed)
- Track and display remaining incorrect guesses
- End the game with a win or lose message and reveal the correct word

### 🛠️ (Optional) Extras

#### Description
Add polish and extended features to make the game more engaging.

#### Requirements (optional)

- Add difficulty levels that change the word list or number of attempts
- Support multi-word phrases
- Keep a simple high-score table or attempt counter

## 📁 Starter files

- `starter-code.py` — minimal scaffold to help you get started (see this folder)

## ▶️ How to run

Run the game with Python 3:

```bash
python3 starter-code.py
```

Place your completed `hangman.py` (or modify `starter-code.py`) in this folder and update the `README.md` with any notes about design decisions or extra features you added.
